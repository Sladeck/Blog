Il n'y a malheureusement pas d'identifiants de connexion car la connexion ne fonctionne pas.

Pour changer directement vos identifiants, veuillez vous rendre dans la base de données.

(Attention, tout les mots de passes doivent être hasher, ici, ils sont en sha1).

BDD -> SUPROCK

TABLE -> USER :
-ID_USER	-> INT
-NOM_USER	->VARCHAR
-PRENOM_USER	->VARCHAR
-PSEUDO_USER	->VARCHAR
-MDP_USER	->VARCHAR
-MAIL_USER	->VARCHAR
-DROIT_USER	->INT


TABLE -> ARTICLES :
-ID_ARTICLE		->INT
-TITRE_ARTICLE		->VARCHAR
-DATE_ARTICLE		->DATE
-CONTENU_ARTICLE	->TEXT

TABLE -> COMMENTAIRES :
-ID_COMM	->INT
